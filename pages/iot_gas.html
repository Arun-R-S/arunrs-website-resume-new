IoT Based Gas Monitoring System

PHP, One Signal Notfication API, PhpMyAdmin(SQL), Arduino

In India most of the household appliances include the LPG gas cylinder for cooking. In day-to-day life the need LPG cylinder increases widely. But most of the users doesnâ€™t have the awareness about the leakage of the gas. The gas leakage happens due to the use of old gas tubes, worn out regulators, improper cylinder knob, etc. The LPG gas contains the mixture of butane and propane. If the 1.8% to 9.5% of the gas is mixed with the air will make it an explosive. So small leakage causes a wide range of damage when burst. So, this device will detect the leakage of gas and alert the house owner immediately. This intimation will prevent from the explosion.

The gas sensor detects the gas mixture which present in the air. If the gas mixture detected it turn on the alarm which alerts the nearby housemates. This can be completely monitored by an IoT automation website. We have developed our own IoT website which bring automation to the gas monitoring system. It also sends a push notification to the user mobile which helpful to alert the user remotely. And the status of the system is completely monitored by the IoT server. The NodeMCU which continuously feed the data to the server which make this an IoT based project. The notification alerts the user and prevents the further caution. This device is portable so it can be installed easily.
